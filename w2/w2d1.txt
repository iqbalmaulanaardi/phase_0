Tugas nomor 2

Algoritma:
1. input tahun("year")
2. jika nilai "year" modulo 4 sama dengan nol, maka jika nilai "year" modulo 100 sama dengan 1, maka tampilan Tahun "year" adalah tahun kabisat, jika tidak maka jika nilai "year" modulo 100 sama dengan 0 dan nilai "year modulo 400 sama dengan 0, maka tampilkan Tahun "year" adalah tahun kabisat jika tidak maka tampilkan Tahun "year" bukan tahun kabisat.

Pseudocode:
READ and SAVE "year"
IF ("year" modulo 4 == 0)
	DO IF("year" modulo 100 == 1)
		DO DISPLAY (Tahun "year" adalah tahun kabisat)
	DO ELSEIF ("year" modulo 100 == 0 AND "year" modulo 400 == 0)
		DO DISPLAY (Tahun "year" adalah tahun kabisat)
ELSE
		DO DISPLAY (Tahun "year" bukan tahun kabisat)
	
Tugas nomor 3

Algoritma:
1. simpan nilai "jumlah_pakaian" dengan angka 20
2. simpan nilai "hitung_pakaian" dengan angka 0
3. ulangi step 4 selama jumlah pakaian tidak sama dengan 0
4. Kurangi nilai "jumlah_pakaian" dengan 1 dan tambah "hitung_pakaian" dengan 1
5. tampilkan nilai "hitung_pakaian"

Psedocode:
STORE "jumlah_pakaian" to 20;
STORE "hitung_pakaian" to 0;
WHILE "jumlah_pakaian" != 0
	DO minus "jumlah_pakaian" by 1 AND add "hitung_pakaian" by 1
ENDWHILE
DISPLAY "hitung_pakaian"

2. ulangi step 3 selama nilai "jumlah_siswa" tidak sama dengan 0
3. Jika nilai "kuku_panjang" sama dengan true maka tampilkan "guru menghukum siswa/siswi", jika tidak maka tampilkan "guru memuji siswa/siswi"

Pseudocode:
STORE "jumlah_siswa" to 40
STORE "kuku_panjang" with any value
WHILE "jumlah_siswa" != 0
	IF ("kuku_panjang" == true)
		DO DISPLAY "guru menghukum siswa/siswi"
	ELSE 
		DO DISPLAY "guru memuji siswa/siswi"
	
	DO minus "jumlah_siswa" by 1
